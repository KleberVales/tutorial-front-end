# Tutorial front-end

HTML: Estrutura e Conteúdo 

1. Introdução ao HTML
   - O que é HTML?
   - Estrutura básica de um documento HTML.
   - Tags essenciais: < html >, < head >, < body >.
  
2. Elementos e Tags Comuns
   - Títulos: < h1 > a < h6 >.
   - Parágrafos: < p >.
   - Links: < a >.
   - Imagens: < img >.
   - Listas:
     * Não ordenadas: < ul > e < li >.
